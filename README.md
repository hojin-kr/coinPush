# Coin Push Bot (코인푸시봇)
코인 현황에 대해 주기적으로 체크해서 푸시 
- 푸시 대상 : 트위터
- 서버 : Github Action (Cron 스케쥴링)
https://twitter.com/coinPushBot

#### 참고 

cctx 라이브러리

#### 내용 

코인 현황에 대해 주기적으로 체크해서 푸시 

- 거래소별 거래 정보 파싱 (cctx)
- 분석
- 이미지로 제작 (Text to Image)
- 트위터로 게시 (twurl)

#### 분석
- 최고 변동률
- 가장 큰 볼륨
- 볼륨 점유율
