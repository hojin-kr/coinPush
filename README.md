# Coin Push Bot (코인푸시봇)
코인 현황에 대해 주기적으로 체크해서 푸시 
- 푸시 대상 : 트위터
- 서버 : Github Action (Cron 스케쥴링)

#### 참고 

cctx 라이브러리

#### 내용 

코인 현황에 대해 주기적으로 체크해서 푸시 

- 거래소별 거래 정보 파싱 (cctx)
- 분석
- 이미지로 제작 (Text to Image)
- 트위터로 게시 (twurl)
